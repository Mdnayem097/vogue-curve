# VogueCurve - Premium Fashion Storefront

VogueCurve is a modern, performance-first e-commerce storefront designed for curated clothing collections. Built with Next.js 15, React 19, and Tailwind CSS, the platform delivers an intuitive shopping experience featuring dynamic catalog structures, global cart management, interactive product configuration, and fluid micro-interactions.

## Core Features

- **Curated Catalog UI**: Highly responsive apparel display optimized for category browsing (Panjabi, Saree, Kurti, Shirts).
- **Dynamic Routing**: Implementation of Next.js App Router mechanics for fast server-side rendered product profiling.
- **Stateful Cart Management**: Centralized React Context API architected to process dynamic selections, scale item volume, validate metrics, and compute costs.
- **Smart Product Validation**: UX workflow preventing cart updates without configuration variables (e.g., mandatory size constraints before addition).
- **Modern Layout System**: Pure Tailwind utility composition leveraging custom gradient overlays, interactive group states, and responsive media handling.
- **Actionable Call-to-Actions**: Minimalist marketing utilities including functional newsletter capture setups with optimized interactive validation.

## Tech Stack

- **Core Engine**: Next.js 15 (App Router)
- **UI Architecture**: React 19
- **Design System**: Tailwind CSS
- **Icon Set**: Lucide React
